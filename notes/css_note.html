布局 ———————————————————————————————— display:block; 表示块级元素 块级元素会自动在前面和后面加上换行，并且在其上的width和height也能够生效 div默认是块级元素 span默认是内联元素(不会有换行,width和height也不会生效) display:inline; 表示内联元素 内联元素前后没有换行，并且在其上的width和height也无效。 其大小由其中的内容决定 div默认是块级元素 span默认是内联元素 内联是不换行，但是不能指定大小 块级时能制定大小，但是会换行 有时候，需要元素处于同一行，同时还能指定大小，这个时候，就需要用到内联-块级 inline-block 两种方式都可以实现水平摆放，主要区别 1. float 使用结束后需要清除浮动，inline-block不需要 2. float 在元素高度不一致的情况下会出现包围现象， inline-block没有 3. float 可以做到靠左 靠右， inline-block只能靠左 --------------------------------- overflow: hidden; 隐藏溢出部分。 结合图片div的float:left,就可以做到基本信息div和图片div水平摆放，并且基本信息div自动占用剩下所有的宽度
